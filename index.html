Croissant

A dumb game for smartypants startuppers.

Which of your friends eats startup for breakfast? Think you’re a fully signed up tech scenester? Put your pastry where your mouth is & prove it...

You will need:

  • Some people. Friends, ideally
  • Overconfidence in your startup and tech knowledge
  • Maybe some beverages

How to play

  • Say the name of something in the tech scene
  • Point to a friend using your non-dominant hand
  • They follow by saying something that starts with the last letter of the preceding name
  • Repeat until someone takes too long or uses the wrong hand
  • Shout "croissant" and start again

Wait. Did you get that? Each word starts with the last letter of the preceding word. For instance "Yahoo!" => "Oculus" => "Silicon Valley" => "Yelp" => "Err..." => "Croissant!".

Awesome. You’re all set.
